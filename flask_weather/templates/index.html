About This Project

This Weather App is a Flask-based web application that allows users to check the weather for multiple cities. It uses the OpenWeatherMap API to fetch real-time weather data.

Technologies Used:

  • Python
  • Flask (Web framework)
  • Flask-SQLAlchemy (Database ORM)
  • Requests (HTTP library for API calls)
  • HTML/TailwindCSS (Frontend)
  • Alpine.js (JavaScript framework for interactivity)

Key Features:

  • Add multiple cities to track weather
  • Display temperature, weather description, and icon
  • Responsive design for mobile and desktop
  • Smooth transitions and hover effects
